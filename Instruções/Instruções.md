***O que é o Node.js?***



&nbsp;	O Node.js é um ambiente de execução que permite executar código JavaScript fora do navegador. Em vez de o JavaScript rodar apenas em páginas web, o Node.js permite usá-lo para criar servidores, APIs, ferramentas de desenvolvimento e aplicações de linha de comando.





***O que é o Vite?***



&nbsp;	O Vite é uma ferramenta de desenvolvimento para aplicações front-end. Ele serve para criar e executar projetos usando frameworks como React, Vue, Svelte, Solid e até projetos em JavaScript puro.
