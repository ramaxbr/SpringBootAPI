\# 🎬 Cinema API



Projeto desenvolvido durante os estudos no \*\*SENAI\*\* com o objetivo de aplicar conceitos de desenvolvimento de APIs REST utilizando \*\*Java\*\* e \*\*Spring Boot\*\*, integrando banco de dados PostgreSQL, autenticação JWT e serviços da AWS.



\## 🚀 Tecnologias Utilizadas



\### Back-end

\- Java 21

\- Spring Boot

\- Spring Security

\- JWT

\- Spring Data JPA

\- Hibernate

\- Maven



\### Banco de Dados

\- PostgreSQL



\### Infraestrutura

\- AWS Elastic Beanstalk

\- Amazon RDS

\- Amazon S3



\### Ferramentas

\- Docker

\- Docker Compose

\- Git

\- Git Bash

\- Node.js

\- Vite

\- IntelliJ IDEA

\- Swagger/OpenAPI

\- Postman



\## 📋 Funcionalidades



\- Autenticação com JWT

\- Controle de acesso (ADMIN e CLIENTE)

\- CRUD de Filmes

\- CRUD de Salas

\- CRUD de Sessões

\- Cadastro de Usuários

\- Sistema de Reservas

\- Upload e remoção de posters no Amazon S3



\## 🏗️ Arquitetura



O projeto foi desenvolvido utilizando arquitetura em camadas:



\- Config

\- Controller

\- DTO

\- Model

\- Repository

\- Security

\- Service



\## 🐳 Docker



A aplicação foi preparada para execução utilizando \*\*Docker\*\* e \*\*Docker Compose\*\*, facilitando a configuração do ambiente de desenvolvimento.



\## ☁️ Deploy



A API foi publicada na AWS utilizando:



\- Amazon RDS para o banco de dados PostgreSQL;

\- Amazon S3 para armazenamento dos posters;

\- AWS Elastic Beanstalk para hospedagem da aplicação.
